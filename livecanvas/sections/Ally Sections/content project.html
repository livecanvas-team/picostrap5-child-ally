Design Process

Discovery: Conducted stakeholder workshops, user interviews, and data audits to map key pain points and usage patterns.

Architecture: Defined an intuitive content structure and prioritized primary actions for streamlined flows.

Prototyping: Built interactive wireframes in Figma, iterating based on usability feedback and A/B testing metrics.

Testing: Ran moderated sessions and remote unmoderated tests to validate task success rates and satisfaction scores.

Visual Design: Developed a modular component library with brand-consistent typography and color palettes, documented in a comprehensive style guide.
